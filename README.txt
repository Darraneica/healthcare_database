Healthcare Database System
This is a Healthcare Database System designed to manage patient information, including the ability to register new patients, search for existing patients, and view patient details. It features an easy-to-use interface for healthcare professionals to handle patient records effectively.

Features
Dashboard: Displays basic stats like the total number of patients and the latest registered patient.
Register New Patient: A form to register new patients with fields for personal and medical details.
Search Patients: Search patients by name or ID to view their details in a tabular format.
Patient Data: Includes important information like name, patient ID, date of birth, medical history, and billing history.
